Instructions

This is a spicy exercise!

sketch.js contains code to draw a "soft keyboard", a keyboard that appears on screen, much like the keyboard displayed on mobile devices. Your task is to make it possible to type using the soft keyboard. Take a moment to read the code and make sure you understand how it works.

You will need to implement the mouseClicked() event function. Inside the function, use the information in the keys array to figure out which character, if any, the user has typed. Display the typed screen on the canvas as the user types.

When you're done, check your work using check-exercise1-4.html.

Expected output:

To figure out which key has been typed, use a standard for or a for... of loop to iterate through the keys array and check if the mouse coordinates are over a key object. Take note that all keys are the same size, except the space bar, which is wider than the others.

To display the characters the user types, you will need a global variable. Each time a key is typed, concatenate the new key name with the message typed so far. Remember that strings are immutable so you can't directly add a new character to a string variable. You can, however, overwrite a string variable e.g. 

let message = "";
message = message + newCharacter;

Display the global string variable somewhere above the keyboard.
