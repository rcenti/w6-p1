Exercise 1.3

Your sketch:

Instructions

Create a sketch of any size. Display a message on the canvas and animate it so that a new character of the message appears every half a second. When the full message is displayed, return to the start. See the video below for an example.

Style the text however you like.

There are two ways to animate at a lower speed than the standard frame rate of 60 frame per second:

  1. Use frameRate() to run the sketch at a slower frame rate.
  2. Use the frameCount system variable to work out when half a second has elapsed. The enterUsername example from last lecture shows how to do this.

JavaScript provides handy string methods that will be useful for this exercise.

When you're done, check your work using check-exercise1-3.html.

Expected output:

You will need two global variables to store strings. One variable (or constant) should store the complete message and the other should store the portion of the message to display. Alternatively, you can use a number variable to keep count of how many characters to display from the message.

The string.slice() method is useful for this exercise.
