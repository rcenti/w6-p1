Exercise 1.2

Your sketch:

Instructions

Download two fonts of your choosing. You can find free open type fonts at FontSpace and Google Fonts. Put your downloaded font files (just the file ending in .ttf) into the assets folder in this sketch folder.

Create a sketch at a size of your choosing and load both fonts into your sketch.

Display a phrase of your choosing using one of the downloaded fonts. When the user presses "f", swap the font used to display the text.

Try some of the text styling function provided by p5.js, including stroke(), fill(), and any styling functions listed in the Typography section of the p5.js reference.

When you're done, check your work using check-exercise1-2.html.

Expected output:

You will need two global variables to store the loaded fonts.

You will also need to implement the p5.js preload() function. In the function, call loadFont() to load each font file and store them in the global variables. You need to supply loadFont() with the path to a font file. Assuming your fonts are stored in the assets folder, your path will begin with "assets/". Don't forget the file extension, likely .ttf, when specifying the file path.

To switch between the two fonts, you will need a third global variable, which will track which font is active. When the "f" key is pressed, use a simple conditional (if - else) to check which font is active, then switch to the other font. Here is some partial code to help you get started: 

let font1; // A global variable to store the first font
let font2; // A global variable to store the second font 
let currentFont; // A global variable that will track which font is active

function preload() {
    // Call loadFont() to load font 1 and 2:
    font1 = loadFont("assets/your-font-file1.ttf");
    font2 = loadFont("assets/your-font-file2.ttf");

    // don't set currentFont yet!
}

function setup() {
    // create the canvas first...

    // then set currentFont
    currentFont = font1;

    textFont(currentFont); // use the first font when the sketch starts
}

function draw() {
    // Draw the text here...
}

function keyPressed() {
    // check which key is pressed (you will need to fill this in)
    // swap the fonts
    if (currentFont === font1) {
        currentFont = font2;
    } else {
        currentFont = font1;
    }

    // Don't forget to use textFont to set the font...
}
